# Cognexus

An AI powered exam correction platform and RAG-Based Chatbot.

---

## Repositories

| Service                                                                         | Description                                                                 |
| ------------------------------------------------------------------------------- | --------------------------------------------------------------------------- |
| [ocr-engine](https://github.com/MedAmineBada/Cognexus-OCR-engine)               | Extracts and processes text from exam images                                |
| [exam-orchestrator](https://github.com/MedAmineBada/Cognexus-exam-orchestrator) | Controls the end-to-end flow of exam creation, correction, and grading      |
| [anti-cheat](https://github.com/MedAmineBada/Cognexus-anti-cheat)               | Detects suspicious behavior and potential cheating during exams             |
| [document-parser](https://github.com/MedAmineBada/Cognexus-document-parser)     | Extracts text from PDF documents                                            |
| [external-gateway](https://github.com/MedAmineBada/Cognexus-external-gateway)   | Handles all communication with external APIs, including LLMs and Cloudinary |
| [api-gateway](https://github.com/MedAmineBada/Cognexus-API-Gateway)   | Handles all incoming requests, managing authentication and authorization |
